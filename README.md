# Python Visualization Hub

Static GitHub Pages hub for classroom Python visualizations.

Public page:
https://artegas.github.io/python-symbol-square-visualizer/

## Structure

- `index.html` - visualization lobby.
- `visualizations/symbol-square-3x3/` - nested-loop visualizer for searching a 3x3 square of matching symbols.
- `visualizations/drone-extraction/` - two-step visualizer for turning drone file rows into `moves`, then accumulated route `points`.
- `visualizations/drone-gcd-31/` - step-by-step visualizer for task 3.1: `abs(A)`, `abs(B)`, `gcd(a, b)`, and `count`.
- `visualizations/drone-task-32a/` - step-by-step visualizer for task 3.2a: counting route points strictly inside the 5000x5000 square.
- `visualizations/drone-task-32b/` - step-by-step visualizer for task 3.2b: finding a midpoint triple with parity checks and `point_set`.
- `visualizations/drone-task-32/` - combined 3.2 page kept locally as a fallback/overview, not linked from the lobby.
